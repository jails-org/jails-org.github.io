## Instalation

Jails is available via [npm]() or [yarn]().

    npm install jails-js -D

You can also use it directly from [cdn]().

    <script type="text/javascript" href="https://cdn.jsdelivr.net/npm/jails-js@3.3.4/dist/jails.min.js"></script>
