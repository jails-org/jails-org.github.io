    • Packages

    The Packages repository was created in order to help you to scale Jails as your application grows in complexity.

    Jails-org/Packages is a realm containing a curated list of great packages found on github plus some official packages developed for Jails.

    Official Packages

    • Logger : Jails Logger
    • Store : State Management
    • Reactor: Template System
    • Arch : Application Architecture
    • jQuery.adapter : jQuery Event System
